-Unit test: Checks a small single part of the code in order to ensure that it works as expected. Its used for verify each function is working the way it was designed to. 
-Integration test: Checks bigger and multiple functions that they work otgether as entire unit or whole. Making sure the system works when they interact with each other. 
-Regression test: When a bug is found and fixedd, you register and fix it and include a test so that the same bug doesnt occur again.
